# 🧊 Arch Linux BSPWM Setup (Daily Arch Warrior)

> Minimal. Fast. Clean. No bullshit.

A fully automated Arch Linux setup using **bspwm + sxhkd**, built for daily use with a clean Nordic aesthetic.

---

## 🚀 Features

- ⚡ Minimal Xorg + BSPWM setup
- 🎹 Windows-friendly keybinds (muscle memory safe)
- 🎨 Nordic theme (GTK + Icons + Terminal + Polybar)
- 📊 Polybar (workspace, audio, battery, clock)
- 🌫️ Picom (blur, shadow, vsync)
- 🧠 Auto workspace rules (code / web / files)
- 🔁 Idempotent install (run multiple times safely)
- 🐚 Zsh + aliases + clean prompt
- 🔥 One command install
